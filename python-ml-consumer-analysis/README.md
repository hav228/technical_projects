Consumer Purchasing Patterns Analysis Program	Lehigh University
Python (pandas, NumPy, scikit-learn, Matplotlib, Seaborn)	ENGR 010 Final Project
Developed machine learning program analyzing 25,000+ consumer transactions using K-means clustering, decision tree classification, and Linear Regression for data analysis and forecasting.
Engineered temporal and consumer features (spending trends, transaction frequency, return frequency) achieving predictive accuracy of over 85% for customer behavior modeling.
Created comprehensive visualizations and statistical reports using Matplotlib and Seaborn for actionable business insights based on purchasing trends in an easily digestible format.
