Activity Based Playlist Creator 
MySQL, Database Design, SQL Queries 
Designed and implemented a relational database system using musical data (BPM, valence, energy level) to connect songs, activities, and genres using many to many relationships and 4 table schema. 
Included 10 complex SQL queries with correlated subqueries, aggregations, Multi-table joins, and Views to analyze the compatibility of songs and activities and form personalized playlists. 
Developed an algorithm for recommendations with scoring system based on BPM and energy level.
